# QA API Testing Project

## Project Overview

This project demonstrates API testing using Postman, including validation, test case design, and bug reporting.


## Tools Used

* Postman
* Excel (Test Cases)
* Word (Bug Reports)


## APIs Tested

* Get All Products
* Get Single Product
* Search Products
* Invalid Product (Negative Testing)


## Validations Performed

* Status Code Validation
* Response Time Check
* Data Validation
* Error Handling


## Bugs Identified

* Performance delay in API response
* Unclear error messages for invalid requests


## Project Structure

/api-tests → Postman collection
/test-cases → Test case documentation
/bug-reports → Bug reports
/screenshots → Proof of execution
